Регистрация контрагентов физ. лиц

Приложение предназначено для регистрации, просмотра и изменения карточки контрагента физ. лица.

Функциональность

В верхней части приложения расположены две кнопки: «Счета клиента» и «Зарегистрировать» (при регистрации нового клиента)

или «Сохранить» (при просмотре существующего)

Кнопка «Зарегистрировать» («Сохранить») будет недоступна пока в данный контрагента не внесены какие-либо изменения.

Все реквизиты клиента поделены на закладки (перемещаться по ним можно с помощью клика на соответствующий ярлычок): «Основные реквизиты», «Реквизиты налогоплательщика», «Экономические нормативы», «Реквизиты клиента», «Дополнительная информация», «Дополнительные реквизиты» и «Доверенные лица».

Закладка «Основные реквизиты»

На этой закладке есть 3 поля необходимых для заполнения, они отмечены красными звездочками. Все 3 поля наименования клиента автоматически переводятся в верхний регистр сразу после ввода. Для того чтоб ввести адрес клиента физ. лица необходима нажать на кнопку справа от поля "Адрес клиента"

После чего на экране появится окошко для ввода полного адреса клиента.

При вводе кода ОКПО программа автоматически проверяет его правильность и наличие уже зарегистрированных пользователей с таким же ОКПО. В случае если введенное ОКПО найдено в базе, пользователю будет предложено перейти на карточку найденного контрагента или ввести другой ОКПО.

Код ОКПО 0000000000 не проверяется (используется для регистрации лиц-нерезидентов которые не имеют идентификационного кода)

Закладка «Реквизиты налогоплательщика»

Реквизиты налогоплательщика необязательны для заполнения, для этого нужно убрать галочку "Заполнять реквизиты налогоплательщика".

При выборе областной НИ, в список районных НИ автоматически подтягиваются соответствующие значения.

На данной закладке обязательными для заполнения полями являются "Дата рег. в НИ" и "Дата рег. в Адм.". Это поля типа даты и должны заполнятся в формате dd.mm.yyyy.

Закладка «Экономические нормативы»

Экономические нормативы необязательны для заполнения, для этого нужно убрать галочку "Заполнять Экономические нормативы".

На данной закладке все поля являются обязательными для заполнения. В полях ввода стоит ограничение на количество вводимых символов. Можно вводить сразу числовые значения нормативов

или же выбирать их из справочника, который всплывает по нажатию на кнопку в поле описания.

Если введенного пользователем кода нет в справочнике, то в поле описания появится значение "Not found".

Закладка «Реквизиты клиента»

Реквизиты клиента необязательны для заполнения, для этого нужно убрать галочку "Заполнять Персональные реквизиты".

Обязательными для заполнения являются поля "Номер док.", "Кем выдан" и "Когда выдан". Поле "Кем выдан" может заполняться из справочника.

Закладка «Дополнительная информация»

Поле Менеджер клиента (код) можно заполнять или вводя код или выбрав его из справочника,

для этого необходимо нажать кнопку слева от поля ввода.
