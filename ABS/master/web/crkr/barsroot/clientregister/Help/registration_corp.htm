Визирование (список документов ожидающих визирования)

Приложение предназначено для просмотра, визирования, сторнирования документов ожидающих визирования в данной группе.

Функциональность

В верхнем левом углу указано название группы контроля под которой вошел пользователь.

Во второй строчке видны все функциональные кнопки (см. ниже "Описание кнопок")

Далее видим таблицу с данными. Детальное описание работы с WebGrid см. здесь. В таблице в рядках помещены данные про документы которые стоят в очереди на визирование. Для выполнения каких либо действий с документом необходимо выделить его в первом столбике таблицы. Отмечать можно сразу несколько документов. Для того чтоб отметить сразу все видимые документы кликните на значок в верхнем левом углу таблицы.

Вторая колонка содержит в себе референсы документов, которые являются ссылками на их карточки. См. приложение "Просмотр карточки документа"

Колонка "Назначение платежа" показывает, только 25 символов реального назначения платежа. Для того чтоб увидеть полную запись, необходимо подвести курсор мыши к этому полю нужного документа и подождать 2 сек. Полный текст всплывает в окошке т.н. тултип.

Сразу же после выделения документа (клик в любой части строки таблицы, рядок подсвечивается серым цветом) , в нижней таблице мы видим всю историю виз по данному документу.

Верхний рядок содержит ФИО пользователя который ввел данный документ. Далее идут следующие по маршруту визирования визы с отметками их наложения и ФИО пользователей ставивших отметки.

Результаты действий пользователя (визирование, сторнирование, возврат на одну) отображается в отдельном окошке с описанием каждого из документов. Напр.:

Данная ошибка возникает, когда некоторые из документов, которые пользователь пытается завизировать, введены этим пользователем, то есть не могут быть им завизированы.

Данное сообщение пользователь получает в случае удачного выполнения операции.

Описание кнопок

- "Установить фильтр" см. компонент WebGrid

- "Обновить данные"
Если данные были изменены (напр. один из документов завизирован другим пользователем) во время работы пользователя другим приложением, то изменения не будут отображаться пока не будет нажата кнопка "Обновить данные"
- "Виза"
По нажатию на кнопку "Виза", все отмеченные документы будут визированы. Для того чтоб отметить документ нужно в таблице выбрать строчку необходимого документа и поставить отметку в левой колонке.
- "Сторнировать"
Сторнирует все отмеченые документы
- "Возврат на одну визу"
Функция "Возврат на одну визу" удаляет последнюю наложенную на документ визу. Если документ еще не имеет виз, то операция не внесет никаких изменений.
