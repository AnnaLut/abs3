Справочная информация о странице Вверх

Выбор даты

Страничка предназначена для выбора даты или периода дат для отображения списка документов.

Функциональность

Кликнув на ссылку в первой строке таблицы

пользователь перейдет на просмотр документов введенных сегодня (автоматически подставляется системная дата).

Во второй строчке пользователь может указать дату.

Тогда будут отображаться документы введенные за период от указанной до текущей системной дат.
