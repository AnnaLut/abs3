Просмотр документов отделения

Приложение предназначено для просмотра документов и их карточек, введеных в текущем отделении.

Функциональность

В левом верхнем углу находится кнопка "Перечитать" см. ниже.

В следующей строчке указано текущее условие фильтрации

Для работы с фильтром используйте следующую строку.

В левом выпадающем списке

пользователь выбирает поле для фильтрации.

В среднем выпадающем списке

пользователь выбирает операцию фильтрации. (прим. предустановленной является «похож». Операция «похож» не совместима с целочисленными полями)

В поле "Значение"

пользователь вводит значение для отбора. Для обозначения нескольких любых символов подряд (только в операции "похож") используется символ "*" (напр. при фильтрации с условием Счета отправителя похож на "1*9", пользователь увидит все счета типа 129, 146569, 1654659) . Для обозначения одного любого символа (только в операции «похож») используется символ "?" (напр. при фильтрации с условием Счета отправителя похож на "1?9", пользователь увидит все чета типа 129, 159, 179).

После того как пользователь выбрал все необходимые значения нужно нажать кнопку «Принять»

После этого выбранное пользователем условие будет добавлено к ранее установленному с префиксом «И».

Для удаления фильтра необходимо нажать кнопку "Отмена" (фильтр удаляется полностью).

Данный фильтр не предусматривает удаление одного условия отдельно!

Под фильтром находится таблица с данными (объект WebGrid). Детальное описание работы с WebGrid см. здесь.

Кликнув на референсе необходимого документа

в первой колонке таблицы, пользователь переходит на приложение просмотра карточки документа.

Описание кнопок

- "перечитать". Обновляет данные таблицы. Например, если у состояние документа изменилось (его сторнировал другой пользователь и он должен отображаться красным цветом) в то время когда открыто приложение, то пользователь увидит изменения только после нажатия на кнопку «Перечитать».
