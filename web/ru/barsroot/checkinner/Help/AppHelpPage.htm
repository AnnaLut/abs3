Визирование

Приложение предназначено для визирования введенных документов.

Функциональность

Приложение содержит в себе два вида визирования: визирование “своих” операций, визирование "общее"(по группам выданным пользователю)

Администрирование пользователей (выдача им групп визирования) производится из комплекса.
