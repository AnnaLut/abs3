Визирование (выбор группы контроля)

На данной этапе пользователь может выбрать одну из доступных ему групп контроля для последующего визирования от ее имени.

Функциональность

Пользуясь мышкой или клавиатурой выберите необходимую группу контроля, выделив ее в списке. Нажмите кнопку "Выбрать", для перехода на страничку перечня документов требующих визирования в данной группе.

Дополнительно

В списке пользователь видит все данные ему группы контроля кроме "самовизы". Если пользователю не имеет ни одной группы, то список будет пустым.
