Справка BARSWEB (изменение текущее отделение)

Страница предназначена для установки текущего отделение пользователя.

Функциональность

С помощью основного комплекса, пользователю выдается список допустимых для работы отделений. Так как on-line соединение не позволяет привязать пользователя к конкретному месту, пользователю необходимо установить текущее отделение при первом использовании BARSWEB. Все последующие разы отделение будет устанавливаться автоматически.

На страничке, отображаются все доступные пользователю отделения. Для выбора необходимого отделения нужно с помощью мыши выделить строчку

и нажать кнопку

"Выбрать"
