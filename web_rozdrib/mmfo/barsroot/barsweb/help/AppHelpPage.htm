Справка BARSWEB

Общая оболочка для всех веб-приложений (веб-комплекс). Содержит в себе весь набор доступных пользователю функций, инструментарий и тп.

Функциональность

Приложение состоит из трех частей:

"Заголовок BARSWEB"

Верхняя панель предназначена для быстрого доступа к функциональным приложениям комплекса. С помощью кнопки

в верхнем левом углу, пользователь может прятать и восстанавливать меню армов.
В окошке «Отделение»

пользователь может увидеть название отделения, в котором он сейчас зарегистрирован. Также пользователь может изменить текущее отделение нажав на кнопку

справа от окошка.

В следующих окошках можно увидеть имя текущего пользователя

и банковскую дату

(банковская дата синхронизируется каждые 15 секунд).

Так же в верхней панели есть набор стандартных функций

таких как: «Просмотр счетов пользователя», «Просмотр документов пользователя», «финансовый калькулятор» и тп., их функциональность имитирует закладку «Инструментарий» в комплексе «Барс Millennium».

"Меню армов"

В меню армов пользователь видит доступные ему армы и функции в них. Для того чтоб посмотреть содержимое арма, необходимо кликнуть на кнопочке

справа от имени арма.

В центральном окне загружаются выбранные пользователем приложения.
